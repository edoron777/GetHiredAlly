## 1. Overview

### Purpose

The Home Page is the main dashboard where users see all available services. It should:
- Clearly show the user journey (lifecycle of job search)
- Make each service's purpose immediately clear
- Invite users to start using services
- Provide quick access to tutorial videos

### Key Design Principles

| Principle | Implementation |
|-----------|----------------|
| **Clear Journey** | Two distinct sections representing job search phases |
| **Inviting Language** | Titles and descriptions that speak to user benefits |
| **Visual Hierarchy** | Important services prominent, coming soon services subtle |
| **Consistency** | All cards follow same layout structure |
| **Accessibility** | Video tutorials for each service |

## 7. Video Modal Component

**Desired Behavior:** Video should:
1. Open in a modal (lightbox) centered on screen
2. Cover the existing content (dark overlay)
3. Maintain 16:9 aspect ratio at ALL sizes
4. Expand to larger size while keeping correct proportions
