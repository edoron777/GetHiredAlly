**Primary Button - CRITICAL STYLING:**

THIS IS VERY IMPORTANT. Use INLINE STYLES to force the color:
```jsx
<button
  style={{
    backgroundColor: '#1E3A5F',
    color: 'white',
    padding: '12px 32px',
    borderRadius: '8px',
    border: 'none',
    fontWeight: '600',
    fontSize: '16px',
    cursor: 'pointer',
    opacity: isDisabled ? 0.5 : 1
  }}
  disabled={isDisabled}
>
  Generate My Prep Report
</button>
```

- Text: "Generate My Prep Report"
- Background: #1E3A5F (MUST use inline style, not CSS class)
- Color: white
- Font weight: 600
- Padding: 12px 32px
- Border radius: 8px
- Border: none

**Disabled state:**
- Same background color (#1E3A5F) but opacity: 0.5
- Cursor: not-allowed
