### TASK 2: Expand Supported File Formats + Smart Detection

CURRENT SUPPORTED: PDF, DOCX, TXT

ADD SUPPORT FOR:
- .md (Markdown)
- .rtf (Rich Text Format)
- .odt (OpenDocument - LibreOffice)
- .doc (Old Word format)

UI CHANGE - Show Supported Formats:
Add a small note below the upload area:
"Supported formats: PDF, DOCX, DOC, TXT, MD, RTF, ODT (Max 10MB)"

SMART DETECTION (Nice to Have):
If file extension is unknown or missing:
1. Try to detect file type from content/magic bytes
2. Try to extract text anyway
3. Only show error if content cannot be read

Example: User uploads "my_resume" (no extension) but it's actually a Word doc - try to read it anyway.

LOCATION: 
- Frontend: CVOptimizerPage.tsx (UI message)
- Backend: backend/app/cv.py (file handling)
