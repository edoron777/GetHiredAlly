## TASK: Create Aesthetic PDF Document with Modern Styling

### OVERVIEW

Upgrade the PDF export to look professional, modern, and impressive using ReportLab.

### STEP 1: Font Settings

Use Helvetica font throughout (ReportLab built-in, similar to Arial):

Title:
- Font: Helvetica-Bold
- Size: 26pt
- Color: #1E3A5F (navy)

Section Headers (H2):
- Font: Helvetica-Bold
- Size: 14pt
- Color: #1E3A5F (navy)

Sub-headers (H3):
- Font: Helvetica-Bold
- Size: 12pt
- Color: #374151 (dark gray)

Body Text:
- Font: Helvetica
- Size: 11pt
- Color: Black
- Line height: 1.4

### STEP 2: Add Header Banner

At the top of the first page, add a colored header bar:

- Full page width
- Height: 60pt
- Background color: #1E3A5F (navy)
- Text: "GetHiredAlly" 
- Text color: White
- Font: Helvetica-Bold, 18pt
- Subtitle: "Job Analysis Report"
- Subtitle color: #93C5FD (light blue)
- Font: Helvetica, 12pt

### STEP 3: Add Document Info Box

Below the header banner, add an info section:

- Background: #F3F4F6 (light gray)
- Border: 1pt solid #E5E7EB
- Border radius: 4pt (rounded corners if possible)
- Contains:
  - Job Title: [extracted title] - Bold
  - Company: [extracted company]
  - Generated: [current date and time]
- Font: Helvetica, 10pt
- Padding: 12pt inside box
- Margin below: 20pt

### STEP 4: Section Styling

For each major section (H2 headers):

- Draw a thin line above: color #E5E7EB, 0.5pt
- Space before header: 16pt
- Header text: Helvetica-Bold, 14pt, Navy color #1E3A5F
- Space after header: 8pt
- Optional: Small navy square bullet (▪) before header text

### STEP 5: Bullet Point Styling

For bullet lists:

- Use bullet character: •
- Bullet color: #1E3A5F (navy)
- Text indent: 15pt from bullet
- Space between items: 6pt
- Font: Helvetica, 11pt

### STEP 6: Callout Boxes

Create colored boxes for special content:

Red Flag boxes:
- Background: #FEE2E2 (light red)
- Left border: 4pt solid #EF4444 (red)
- Padding: 10pt
- Margin: 8pt top and bottom

Key Insight boxes:
- Background: #DBEAFE (light blue)
- Left border: 4pt solid #3B82F6 (blue)
- Padding: 10pt
- Margin: 8pt top and bottom

Green Success boxes:
- Background: #D1FAE5 (light green)
- Left border: 4pt solid #10B981 (green)
- Padding: 10pt
- Margin: 8pt top and bottom

### STEP 7: Header on Every Page (except first)

For pages 2 and onwards:

- Small header bar: height 30pt
- Background: #1E3A5F (navy)
- Text: "GetHiredAlly - Job Analysis Report"
- Text color: White
- Font: Helvetica, 10pt
- Aligned left with padding

### STEP 8: Footer on Every Page

- Thin line above footer: #E5E7EB
- Left side: "Generated by GetHiredAlly"
- Right side: "Page X of Y"
- Font: Helvetica, 9pt
- Color: #6B7280 (gray)
- Margin from bottom: 30pt

### STEP 9: Page Setup

- Page size: A4
- Margins: Left 50pt, Right 50pt, Top 50pt, Bottom 60pt
- Background: White

### STEP 10: Remove Raw Markdown

Parse markdown properly - no raw syntax should appear:
- Convert **text** to bold text
- Convert ## Header to styled header
- Convert --- to section spacing (not visible line)
- Convert bullet lists to proper bullets
