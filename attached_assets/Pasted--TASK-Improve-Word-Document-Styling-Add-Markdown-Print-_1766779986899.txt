### PART 2: Add Markdown Download Button

Add a "Markdown" button next to the other download buttons.

When clicked:
1. Take the raw analysis text (it's already markdown)
2. Create a blob with type "text/markdown"
3. Trigger download with filename: "XRay_Analysis_[CompanyName].md"

Code example:
const handleDownloadMarkdown = () => {
  const blob = new Blob([analysisResult], { type: 'text/markdown' });
  const url = window.URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = `XRay_Analysis_${companyName || 'Report'}.md`;
  a.click();
  window.URL.revokeObjectURL(url);
};
