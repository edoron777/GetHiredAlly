DATABASE SCHEMA (cv_issue_types)
───────────────────────────────────────────────────────────────────────

Table: cv_issue_types (57 rows)
- id: integer (auto-increment PK)
- subcategory_id: integer FK → cv_issue_subcategories
- issue_code: varchar (UNIQUE) - e.g., 'CONTACT_MISSING_EMAIL'
- display_name: varchar
- description: text (WHY THIS MATTERS content)
- severity: 'critical' | 'important' | 'consider' | 'polish'
- weight: integer 1-10
- can_auto_fix: boolean
- auto_fix_type: 'full' | 'partial' | 'none' | NULL
- detection_method: 'regex' | 'rule' | 'nlp' | 'ai' | NULL
- static_tip: text (instructions)
- user_effort: 'minimal' | 'medium' | 'extensive' | NULL
- requires_user_input: text
- example_before: text
- example_after: text
- attributes: jsonb
- ui_config: jsonb

Table: cv_issue_categories (7 rows)
- id, code, display_name, icon, color, display_order

Table: cv_issue_subcategories (23 rows)
- id, category_id, code, display_name, display_order

CATEGORIES (7):
1. CONTACT - Contact Information (8 issues)
