  const {

    title,
    description,
    buttonText,
    route,
    videoUrl,
    videoTitle,

          <div className="h-12 flex items-center justify-center">

              <span></span> {/* Empty spacer to maintain height */}
            )}
          </div>
        </div>

## KEY CSS FIXES

| Fix | CSS Class | Purpose |
|-----|-----------|---------|
| Same card height | `h-full` | Card fills container height |
| Content layout | `flex flex-col` | Stack content vertically |
| Description expands | `flex-grow` | Takes available space |
| Button at bottom | `mt-auto` | Pushes to bottom |
| Fixed badge height | `h-8` | Consistent top spacing |
| Fixed video area | `h-12` | Consistent bottom spacing |
| Max card width | `max-w-sm` (384px) | Prevents stretching |
| Center cards | `justify-items-center` | Centers in grid |
