- Pros: Simple fix
- Cons: May miss actual weak usage like "supported the team"

- Keep "supported" in list BUT check context
- Flag: "Supported the team" (passive, no object)
- Don't flag: "Supported [Technical System/Policy/Infrastructure]"
- Pattern: "Supported" + technical noun = VALID

OPTION C - Add to exceptions list:
- Create exception patterns for technical contexts
- "Supported * policies", "Supported * systems", etc.

IMPLEMENTATION:
Choose the option that best fits current code architecture.
If context awareness is too complex, use Option A (remove from list).
