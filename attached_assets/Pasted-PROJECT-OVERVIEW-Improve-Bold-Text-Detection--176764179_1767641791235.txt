Bold text in Word can come from multiple sources:
1. Direct bold on the run (run.bold = True)
2. Bold from font settings (run.font.bold = True)
3. Bold from paragraph style (paragraph.style.font.bold)
4. Bold from character style (run.style.font.bold)
5. Bold inherited from parent styles

The current extraction likely only checks source #1 or #2, missing others.
