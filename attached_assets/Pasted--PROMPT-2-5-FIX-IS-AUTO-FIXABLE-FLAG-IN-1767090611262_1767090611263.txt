The is_auto_fixable flag is FALSE or MISSING on issues.
This causes the after-fix score calculation to return 0 improvement.

Formula: recovery = points_lost * rate * (auto_fixable / total)
If auto_fixable = 0, then recovery = 0

ADD these rules to the prompt for setting is_auto_fixable:

RULES FOR is_auto_fixable FLAG:

Set is_auto_fixable: true for issues that can be fixed by rewriting text:
- Grammar errors (spelling, punctuation) = TRUE
- Weak action verbs = TRUE
- Missing quantification (can add estimates) = TRUE
- Passive voice = TRUE
- Formatting issues in text = TRUE
- Vague descriptions = TRUE

Set is_auto_fixable: false for issues requiring human input:
- Missing contact info (need real data) = FALSE
- Missing work experience (need real data) = FALSE
- Wrong dates (need verification) = FALSE
- Missing skills section (need real skills) = FALSE

DEFAULT: If unsure, set is_auto_fixable: true

For each issue, set "is_auto_fixable" based on these rules:

TRUE (can be fixed by AI):

- missing entire sections (education, experience)
- incorrect dates or company names
- missing specific certifications
- personal information that only user knows

When in doubt, set is_auto_fixable: true
